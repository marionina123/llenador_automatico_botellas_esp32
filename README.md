# llenador_automatico_botellas_esp32
Sistema embebido basado en ESP32 para el llenado automatico de botellas de 500 ml. Incluye detección de botella por ultrasonido, medición de volumen mediante sensor de flujo y control seguro de bomba. Implementa FreeRTOS, interrupciones y lógica de estados. Proyecto final de Sistemas Embebidos I (IMT-222)
